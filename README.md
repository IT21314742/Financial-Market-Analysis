# 📈 Financial Market Analysis & Interactive Dashboard

## Project Overview
This notebook performs comprehensive financial analysis on tech stocks using Yahoo Finance API. It includes data collection, statistical analysis, risk metrics, and interactive visualizations.

---

## 🎯 Features
- Real-time stock data from Yahoo Finance
- Multiple stock comparison (AAPL, GOOGL, MSFT, AMZN, TSLA)
- Daily returns and cumulative performance
- Risk metrics (Volatility, Sharpe Ratio, Max Drawdown)
- Correlation analysis
- Interactive candlestick charts
- Complete portfolio dashboard
